# 💼 CareerConnect Frontend

A modern and responsive Job Portal frontend built with React.js. This application allows job seekers and employers to interact through a clean and user-friendly interface.

---

## 🚀 Live Demo

Frontend: https://your-frontend.vercel.app

Backend API: https://your-backend.onrender.com

---

## ✨ Features

- User Authentication
- Browse All Jobs
- Apply for Jobs
- Employer Dashboard
- Post New Jobs
- View Applications
- Responsive Design
- Protected Routes
- Modern UI

---

## 🛠️ Tech Stack

- React.js
- React Router DOM
- Axios
- React Hot Toast
- React Icons
- CSS3
